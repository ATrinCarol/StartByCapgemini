Pipe customizado

Nesse exemplo, customizamos um pipe com a função de multiplicar dois valores.
Lembre-se que para utilizar o pipe é necessário que o valor esteja dentro das 4
chaves referentes da interpolação.

2x2 = {{ 2 | multiplicaPor: 2 }}

3x2 = {{ 3 | multiplicaPor: 2 }}

5x2 = {{ 5 | multiplicaPor: 2 }}

5x5 = {{ 5 | multiplicaPor: 5 }}

3x7 = {{ 3 | multiplicaPor: 7 }}
